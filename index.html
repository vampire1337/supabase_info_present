Supabase

Открытая альтернатива Firebase

Supabase — открытая платформа, предоставляющая набор бэкенд-сервисов для разработки веб- и мобильных приложений. Построена на базе PostgreSQL и включает инструменты для работы с данными.

База данных PostgreSQL

Основана на PostgreSQL как базе данных для хранения и обработки информации.

Аутентификация

Управление пользователями с безопасностью на уровне строк и различными методами входа.

Realtime API

Возможность подписки на изменения в базе данных и получения обновлений.

Автоматические API

Генерация RESTful и GraphQL API на основе схемы базы данных.

Хранилище

Функционал для хранения и обслуживания файлов с настройкой прав доступа.

Безопасность

Механизмы безопасности на уровне строк и граничные функции.
